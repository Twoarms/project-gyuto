# Gyuto

### Instal Guide
#### Strapi (CMS)
```
git clone https://github.com/bouzouitadavid/gyuto.git
cd gyuto
git checkout cms
npm install
strapi start
```
After cloning go to config/environments/development/database.json  
Change the line 8 "uri": "<INSERT_MY_DATABASE_URI>" (Ryver !)

#### Gatsby (Frameworks)

```
cd site
npm install
gatsby develop
```
Go Develop !
